- 서비스 이름: 무장애 맞춤화 관광지 추천 "챗봇 서비스" - 아띠

# 3. 주요 기능
<img width="1143" alt="스크린샷 2024-08-14 오전 8 18 23" src="https://github.com/user-attachments/assets/bc95b1e1-4af9-4d28-9411-92c5128a75a3">

- **설문조사**:
  - 맞춤화를 위한 서비스 선택.

- **AI챗봇**:
  - 사용자에게 AI챗봇을 통해 궁금해하거나 찾는 관광지에 대한 정도 제공.
 
- **지도**:
  - 채팅에서 언급된 관광지를 사용자 중심으로 지도에서 표시.

- **상세페이지**:
  - 특정 관광지에 대한 세부 정보 제공.
